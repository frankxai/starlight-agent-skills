# OpenCode

OpenCode supports a per-skill Markdown convention.

## Install

```bash
cp skills/cosmos/apod-to-short/SKILL.md .opencode/skills/apod-to-short.md
```

If the skill ships `references/`, copy that alongside and link to it from the
skill body.

## Notes

- Discovery is by filename; keep the kebab-case skill name.
- The body format is already compatible — no rewrite needed.
