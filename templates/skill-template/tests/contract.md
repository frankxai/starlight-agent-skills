# Contract test — skill-name-here

A lightweight, human-or-agent-runnable checklist asserting the skill honors its
output contract. (No test runner required; this is a golden-checklist.)

Given the input in [`../examples/example-01.md`](../examples/example-01.md), the
output MUST satisfy:

- [ ] Produces every field named in `manifest.json` → `outputs`.
- [ ] Every factual claim is traceable to the supplied source.
- [ ] Respects the rights / licensing constraints stated in the skill.
- [ ] Voice matches the brand register declared in the skill.
- [ ] Carries (or its parent artifact carries) the "Built on SIP" attestation.
