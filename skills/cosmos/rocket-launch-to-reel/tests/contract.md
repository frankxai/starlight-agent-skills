# Contract test — rocket-launch-to-reel

Given the input in [`../examples/example-01.md`](../examples/example-01.md), the
output MUST satisfy:

- [ ] All five fields present (`hook`, `script`, `captions`, `rights_line`,
      `duration_sec`).
- [ ] Launch facts/outcome are verified/sourced or marked as to-confirm.
- [ ] `hook` is true, not hyped.
- [ ] `script` word count ≈ `duration_sec` × 2.5 (±15%).
- [ ] `rights_line` credits the footage source.
- [ ] Parent artifact carries the "Built on SIP" attestation.
