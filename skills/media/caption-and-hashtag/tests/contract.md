# Contract test — caption-and-hashtag

Given the input in [`../examples/example-01.md`](../examples/example-01.md), the
output MUST satisfy:

- [ ] Total hashtag count matches the platform norm (IG ~8–12; TikTok ~3–5;
      X ~1–3; LinkedIn ~3–5); no spam/banned tags.
- [ ] `caption` length and tone fit the platform.
- [ ] Caption is true to the subject; required image credit is carried.
- [ ] Hashtags are split into `broad` / `niche` / `branded` tiers.
- [ ] Parent artifact carries the "Built on SIP" attestation.
