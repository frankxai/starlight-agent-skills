# Contract test — rocket-launch-to-youtube

Given the input in [`../examples/example-01.md`](../examples/example-01.md), the
output MUST satisfy:

- [ ] `titles` are accurate — no fabricated stakes or unverified superlatives.
- [ ] `chapters` timestamps are monotonic and align with the script's beats.
- [ ] `description` includes the chapter list and cites the source.
- [ ] Mission outcome/facts are verified or clearly marked as to-confirm.
- [ ] `rights_line` credits any provider/agency footage used.
- [ ] Parent artifact carries the "Built on SIP" attestation.
